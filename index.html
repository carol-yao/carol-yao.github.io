Remindr

Summer 2016 | Web Application
Ruby on Rails / JavaScript / HTML / CSS / Twitter API / Gmail API / Twilio

Remindr helps you keep in touch by tracking your last interaction with friends, family or network. Import contacts, retrieve previous messages and set how often you want to keep in touch and we will send you reminders (if you don't contact them right away, we will send an embarrasing message for you!). Through this project, I learned how to work with the Twitter API, set background and scheduled tasks and to work in an agile team.

Snag

Hack the 6ix | API / Chrome Extension
Ruby // Rails // React // JavaScript // Bootstrap

Twitter API // Indico APIs (Emotion, Sentiment & Keyword Analyses)

Developed for the Wealthsimple challenge, Snag is a simple marketing tool for proactive companies to find potential users for their product. Twitter tweets are pulled in, filtered through multiple Indico APIs to find users struggling with their finances for Wealthsimple to approach. Our team created a simple API to fetch these tweets and respond with JSON for our front end to display. Finally, we packed it into a chrome extension to make it easier to use.
